Gallery Page

Technical Description

The gallery page was implemented using a structured HTML layout to display renewable energy images in an organised and interactive format. Semantic elements such as

,
,
,
, and
were used to separate different parts of the page, improving readability and maintainability. The navigation bar is structured using

    elements, providing a consistent user interface across the website.

    CSS was used extensively to create a modern and responsive layout. A grid layout (display: grid) was applied to the .gallery-grid container, using grid-template-columns: repeat(3, 1fr) to organise images into a three-column structure. This ensures a balanced and visually appealing arrangement. Each gallery item (.cell) uses position: relative, overflow: hidden, and border-radius to create card-like components with smooth edges. Hover effects were implemented using transform: scale() and transition properties to provide interactive feedback when users hover over images.

    A modal popup system was implemented using JavaScript to display enlarged images and additional information. When a gallery item is clicked, an event listener updates the modal content dynamically using properties such as textContent and src. The modal is displayed using a CSS class toggle (.active) and styled with position: fixed and a semi-transparent background to create an overlay effect. The modal content (#modalBody) is centred using Flexbox and includes animations such as scaling (transform: scale) for a smooth opening effect.

    Additional interactivity was implemented by dynamically creating control elements using JavaScript. Dropdown menus allow users to change the modal’s colour scheme and font style, demonstrating DOM manipulation and event handling. The layout is responsive due to the use of relative units, grid layout, and the viewport meta tag, ensuring compatibility across different screen sizes.

    Accessibility

    Semantic Structure and Screen Reader Support
    The gallery page uses semantic HTML elements such as <header>, <nav>, <main>, and <section> to create a clear and meaningful structure. This helps screen readers understand the layout and allows users with visual impairments to navigate the page more easily.

    Image Descriptions and Visual Clarity
    All images include descriptive alt attributes, ensuring that users who cannot see images can still understand their content. Proper font sizes, spacing, and colour contrast were used to improve readability and make the page accessible for users with low vision.

    . User Interaction and Navigation
    Interactive elements such as gallery items and the modal popup are designed to be simple and easy to use. The modal includes a clearly visible close button, allowing users to exit without confusion. The design also remains functional without hover effects, supporting users who rely on keyboard navigation.

    Responsive and Device Accessibility
    The gallery layout is responsive due to the use of CSS Grid, flexible units, and the viewport meta tag. This ensures the page adapts to different screen sizes and devices, making it accessible for mobile users and those who zoom or adjust display settings.
